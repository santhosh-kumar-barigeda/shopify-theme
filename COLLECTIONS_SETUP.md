# Collections Setup Guide

Since your products already have tags, create **Automated Collections** in Shopify Admin.

## How to Create
1. Shopify Admin → Products → Collections → **Create collection**
2. Title: Use the name from the table below
3. Collection type: **Automated**
4. Conditions: Select "Product tag is equal to" and enter the tags
5. Save

## Important Notes
- Make sure product tags in your CSV use the **exact same spelling** as the conditions above
- If a tag doesn't exist on any product, the collection will be empty
- For Quick-Fit, the tag in your CSV may be `Quick-Fit` or `Quick Fit` — match exactly
- `Women's` / `Men's` tags use an apostrophe — make sure your CSV has them

## Verification
After creating collections, visit `/collections/{handle}` to verify products appear correctly.
